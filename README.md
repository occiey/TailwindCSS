# TailwindCSS

### ファイル構成

```shell
.
├── README.md # 当ファイル
├── node_modules # 各種インストールパッケージ
├── package-lock.json # インストールパッケージ情報
├── package.json # パッケージのメタデータ管理
├── src # コーディングディレクトリ
│   ├── index.html
│   ├── sass # sass用ディレクトリ
│   └── tailwind.css # tailwind import
└── tailwind.config.js # tailwindCSS設定ファイル
```

### 使い方

以下のコマンドでリアルタイムにSASSとTailwindCSSが更新される。

```shell
$ npm run dev
```
